Data:
- File SSE_slip_relative_to_linear_trend_Nature_TXT.txt contains the Slow Slip Events slip evolution (in mm) for each subfault relative to longterm linear trend. It is a 3339 subfaults x 3384 epochs matrix.
- File timeline_Nature_TXT.txt contains the time in decimal year.
- File Coord_Center_Subfaults_Nature_TXT.txt contains the coordinates (Lon/Lat) of each subfault center.
- File Lon_Fault_Nodes_Nature_TXT.txt contains the longitude of the subfault nodes.
- File Lat_Fault_Nodes_Nature_TXT.txt contains the latitude of the subfault nodes.
